# easy_base

### 介绍

使用业界主流开源项目，快速低成本搭建基础设施。

### 愿景

假定你和你的朋友有一个 good idea...

假定你和你的朋友是 java coder...

假定你们没啥光环 ，无法开局便迈大步向前。

愿这个项目，帮你在几个小时内，快速搞定 "基础设施" 建设。

愿你们的 "小目标" 更加轻量，敏捷。
